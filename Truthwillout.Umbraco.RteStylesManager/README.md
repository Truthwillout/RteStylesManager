# RTE Styles Manager for Umbraco

A visual dashboard for managing Rich Text Editor styles and colours in Umbraco 13.

## Features

	 **Visual Style Management**
- Colour picker for each style
- Live preview of styled text
- Add, edit, and delete styles easily

	 **Organise Your Styles**
- Group styles into categories (Headers, Text, etc.)
- Reorder styles with up/down arrows
- Edit category names directly

	 **Automatic Updates**
- Automatically generates CSS files
- Updates `rte-style-formats.json`
- Syncs with TinyMCE configuration

## Installation

```bash
dotnet add package Truthwillout.Umbraco.RteStylesManager
```

**After Installation:**
1. Restart your Umbraco application
2. Go to Settings → RTE Styles Manager
3. Start managing your styles!

## Usage

1. **Add styles** with color pickers
2. **Organise** into categories
3. **Save** configuration
4. **Restart** Umbraco
5. Styles appear in RTE dropdown!

## Requirements

- Umbraco 13.0+
- .NET 8.0
